# VS Code Extension

A VS Code extension that provides workspace browsing and canvas viewing within the editor.

## Planned Capabilities

- **Workspace tree**: Browse `.carta/` structure in the VS Code sidebar
- **Canvas viewer**: Read-only WebView rendering of `.canvas.json` files
- **Document navigation**: Click cross-references to navigate between docs
- **MANIFEST preview**: Rendered view of MANIFEST.md with clickable refs
