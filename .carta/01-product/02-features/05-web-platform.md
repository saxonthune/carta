# Web Platform

A hosted web client backed by a server that performs git operations on behalf of nontechnical users. Enables real-time collaboration and workspace management without requiring CLI access.

## Planned Capabilities

- **Web-based workspace editing**: Browse and edit `.carta/` workspaces through a browser
- **Git integration**: Server commits changes to git on behalf of web users
- **Real-time collaboration**: WebSocket sync for concurrent editing
- **Canvas editing**: Full canvas editor accessible via the web
- **AI assistant**: Chat-based AI with MCP tool access
