# `.carta/` Workspace Template

This is the `.carta/` workspace format template. Copy this directory as `.carta/` in your project to get started with structured, AI-navigable documentation.

The `00-codex/` title contains the format specification — how the documentation system works, its conventions, and retrieval patterns. The remaining titles (01-context through 04-operations) are empty scaffolds for your project-specific content.

See `MANIFEST.md` for the machine-readable index and `00-codex/01-about.md` for a human introduction.
