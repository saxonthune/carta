# Team Lead

## Persona

A team lead who defines standardized construct schemas so their team models architecture consistently. May also use collaboration features to work on designs with the team.

## Goals

- Define construct schemas that enforce team conventions
- Create schema groups to organize related types
- Share schemas and designs with team members
- Review team members' architectural designs

## Features Used

- doc03.01.05 (Metamap) — visual schema management
- doc03.01.06 (Schema Editor) — creating standardized schemas
- doc03.01.08 (Import/Export) — distributing schema sets
- doc03.01.09 (Collaboration) — real-time co-editing
