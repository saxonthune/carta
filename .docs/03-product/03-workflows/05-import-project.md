# Import a Project

## Steps

1. Click "Import" in the header (or drag-and-drop a .carta file)
2. The Import Preview Modal shows all items from the file
3. Items are categorized: schemas, instances, deployables
4. Each item shows status: conflict (amber), new (green), existing (gray)
5. Select which items to import via checkboxes
6. Click "Import" to apply
7. Selected items are merged into the current document

## Loading an Example

1. Click Settings in the header
2. Select "Load Example"
3. The Examples Modal shows available bundled .carta files
4. Select an example to preview and import
5. The current document is cleared before loading (like Excalidraw)
