# Create a Construct

## Steps

1. Right-click on the canvas to open the context menu
2. Hover "Add Construct" to see available schemas grouped by schema group
3. Click a schema type to create an instance at the clicked position
4. The construct appears with default field values and a generated semantic ID
5. Click the construct to select it, then edit fields in the details panel
6. Optionally assign a deployable via the dropdown in details view

## Alternative: Add Related Construct

1. Right-click an existing construct
2. Select "Add Related" and choose a port + target schema type
3. A new construct is created near the source, auto-connected via the selected port
