# Define a Schema

## Via Construct Editor

1. Right-click canvas in Metamap and select "New Construct Schema"
2. **Basics tab**: Enter display name (auto-generates type ID), pick color, write semantic description, choose schema group, set background color policy and port display policy
3. **Fields tab**: Add fields via "+ Add Field" button (opens modal). Drag fields between tier zones (pill, minimal, details, full) to control display visibility. Edit/delete via icons on field chips.
4. **Ports tab**: Choose default ports or customize â€” set ID, label, port type, position, offset
5. Click "Create Schema" to add the schema to the document

## Via Metamap

1. Switch to Metamap view
2. Right-click and select "New Construct Schema" (opens wizard)
3. After creation, the schema appears as a node in the Metamap graph
4. Drag between schema nodes to create port relationships via the Connection Modal

## Editing

1. Right-click a schema node in Metamap (or access via settings)
2. Select "Edit Schema"
3. The Construct Editor opens pre-filled with existing values
4. Modify across any tab (Basics, Fields, Ports) and click "Save Changes"
