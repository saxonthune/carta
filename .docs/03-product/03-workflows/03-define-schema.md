# Define a Schema

## Via Wizard

1. Right-click canvas and select "New Construct Schema"
2. **Basics step**: Enter type identifier, display name, pick color, write semantic description, choose display field, select schema group, set background color policy and port display policy
3. **Fields step**: Add fields — set label, type, default value, required flag, placeholder, show-in-minimal-display
4. **Ports step**: Choose default ports or customize — set ID, label, port type, position, offset
5. **Review step**: Preview the schema configuration
6. Click "Create" to add the schema to the document

## Via Metamap

1. Switch to Metamap view
2. Right-click and select "New Construct Schema" (opens wizard)
3. After creation, the schema appears as a node in the Metamap graph
4. Drag between schema nodes to create port relationships via the Connection Modal

## Editing

1. Right-click a schema node in Metamap (or access via settings)
2. Select "Edit Schema"
3. The wizard opens pre-filled with existing values
4. Modify and confirm
